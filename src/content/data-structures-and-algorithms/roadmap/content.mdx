# Data Structures and Algorithms

Below you can find the roadmap of my Data Structures and Algorithms (DSA) course, inspired by the [Algomaster.io DSA patterns list](https://algomaster.io/practice/dsa-patterns). 
The arguments are in a specific order to build upon previously covered concepts.
