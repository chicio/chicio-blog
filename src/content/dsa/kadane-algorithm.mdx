# Kadane's algorithm


---

## 1. Introduction
- What Kadane’s Algorithm solves: finding the maximum sum subarray efficiently.
- Key intuition: dynamic programming approach that tracks current maximum and global maximum.
- Time complexity advantage over naive O(n²) approaches.

## 2. The Algorithm
- Step-by-step explanation of the core idea:
  - Maintain `currentMax` for the max ending at current index.
  - Update `globalMax` if `currentMax` is greater.
